Worm Tank - Sensors

The sensors are what the worms use to detect what is around them in the worm tank.

The "senses" of the worm is split into three sections, ahead, +/-45 degrees, left, -45 to - 135 degrees and right, +45 to +135 degrees. An extra 4th sensor is set if there is nothing ahead, left or right.

The sensors are made up of two parts. What the object they can detect is: food, wall or other worms and how they can detect them:
Seen One or more objects are in the section
Next To The next move in that direction will move on to an object
Range There is one or more object within the given range. This sensor can be given multiple ranges in which case the object is marked as seen in the closest range it appears in and not the further away ones.
Count A count of the objects with the given range. This sensor can be given multiple ranges in which case the object is counted in the closest range it appears in and not the further away ones.

The sensors are referred to in other pages by contatenating the object they are looking for with how they can detect them and the range (if applicable).

e.g. FoodSeen is a sensor to detect food in any direction. WormRange10 is a sensor that detects Worms within a range of 10.
