Worm Tank - Run Menu

The Run Menu qives options to change the currenty running simulation. The options are:

Train Select a worm type and start it training
Battle Select two worm types to compete
Arena All the different worm types compete
League All the different worm types battle each other in a league

Train

A window is shown allowing you to choose a worm type to train.

Pressing OK saves the current tank and loads the latest generation of the given worm type.

If the Reset check box is set the worm type is returned to the first generation.

In training mode 100 worms are created for the first generation and then the simulation run to update the worms positions and health.

When the number of worms reduces to 10 a new generation is started.

The new generation consists of the 10 survivors from the previous generation and then 90 new children randomly created based on the 10 survivors.

The worm that had the most energy at the end of the generation is designated the leader. The leader is given more children than the others including some based by averaging or mixing it with the rest of the top ten.

Switch the worm that is training or to a different mode from the Run menu to finish the training of this worm.

Battle

A window is shown allowing you to choose two worm types to battle against each other.

Pressing OK saves the current tank and loads the latest generation of the two worm types to battle.

The First generation check box means the two sides are generated randomly rather than using the latest generation.

In battle mode 10 of each type of worm are created and the simulation run to update the worms positions and health.

The battle is won when there is only one type of worm left in the tank.

When the battle is won a message is shown displaying the results and the last worms are left wandering around behind the message.

Select an option from the Run menu to leave the battle and do something else. Select Battle again from the Run menu to run another battle.

Arena

This switches the worm tank to the battle arena where 10 of each different worm type is added to the tank and left until only one type of worm is left.

When the arena is won a message is shown displaying the results and the last worms are left wandering around behind the message.

Select an option from the Run menu to leave the arena and do something else. Select Arena again from the Run menu to run a new arena.

League

This switches the worm tank to running the next game in the league.

The league is made up of all the worm types and each type will play every other type twice.

A match in the league consists of 10 worms of two different worm types competing against each other.

The match is run for up to four generations. Each generation is ended when there are only ten worms left. The next generation is started by adding ten new worms as children of the ten survivors.

If after a generation there is only one type of worm left the game finishes.

The game is scored with points added for every worm left (up to a maximum of 10) at the end of a generation. If the match finishes before the 4th generation. The winning side gets a bonus 10 points for every generation left.

When the league is won a message is shown displaying the results and the last worms of the last match are left wandering around behind the message.

You can see the current positions of the worm types in the league by using the League option from the View menu.

Select an option from the Run menu to leave the league and do something else. Selecting League again from the Run Menu will start a new season of the league.
