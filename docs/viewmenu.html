Worm Tank - View Menu

The view menu contains options to view more details about the current state of the simulation.

Worm List Shows a window containing a list of the currently surviving worms
League Shows a window containing the current league results
History Shows a window containing the history for the current worm tank
Generations Shows a window containing the generation each worm type has been trained to

Worm List

The worm list shows a list of all the current worm types grouped by their full name and position.

The fields are:
Name The full name of the worm. For most worm types this is just the name of the worm type. But some types (e.g. Mutant) have extra information added to the name.
Position The position the worm was in at the end of the previous generation or what type of child it is.
Colour The colour of the worm in the tank,
Left The number of worms left in the tank. Once this gets to zero the line is removed from the table.

League

This shows the current positions of the various worm types in the league, the result of the last game and what the current/next game will be.

The fields are:
Team The worm type
Played The number of games played
Won The number of games won
Drew The number of games drew
Lost The number of games lost
For Total of points scored in all games
Agnst Total of points scored against this team in all games
Diff The difference between poinst scored for and against
Points The points gained. 2 points for a win, 1 point for a draw.

History

Shows the results of the last ten generations of the current worm tank. This is relevant to the training mode.

The latest generation is shown at the top.

The fields are:
Ticks The duration of the generation
Food left The amount of food left at the end of the generation
Max energy The maximum energy reached during the generation
Leader The position the worm with the most energy at the end of the generation started the generation at.
Colour The colour of the leader

Generations

This shows the current generation for each of the different worm types

The fields are:
Name Name of the worm type
Generation The current generation of this worm type
