Worm Tank - View Menu

The view menu contains options to view more details about the current state of the simulation.

Worm List Shows a window containing a list of the currently surviving worms
League Shows a window containing the current league results

Worm List

The worm list shows a list of all the current worm types grouped by their full name and position.

The fields are:
Name The full name of the worm. For most worm types this is just the name of the worm type. But some types (e.g. Mutant) have extra information added to the name.
Position The position the worm was in at the end of the previous generation or what type of child it is.
Colour The colour of the worm in the tank,
Left The number of worms left in the tank. Once this gets to zero the line is removed from the table.

League

This shows the current positions of the various worm types in the league, the result of the last game and what the current/next game will be.
