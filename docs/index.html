Worm Tank

Worm tank is a simulation of a group of worms feeding in a tank.

The worms are controlled by different brains which get input from one or more sensors that can tell it where food, other worms or walls are with various accuracy.

In normal training mode the worms are left to roam around the tank finding food until all but ten of them die of starvation.

Once down to the last ten worms a new generation is created, which includes the last ten worms with a group of randomly generated children. The worm which has the most energy left, is designated the leader and he gets more children, including two created by mixing it with the other worms in the top ten.

The other modes allow you to pit the worms against each other to see which survives better.

The user interface

  • Run Menu provides options to start the tank running in the various modes
  • View Menu allows extra information to be shown on the tank
